This project implements this [Figma design](https://www.figma.com/file/7qwsWggv9BAxMi2VPhBuPr/Air-(formerly-Dia)?node-id=9138%3A35)

The page adapted to the following screens:
- Large screens 2560px
- The design 1600px
- Desktop 1280px
- Small desktop 1024px
- Tablet 640px
- Mobile (> 320px)

Layout created pixel perfect using HTML and SCSS according to BEM methodology.
